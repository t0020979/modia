## ⚙️ Конфигурация через ENV

Для управления логированием в разных окружениях используйте глобальные переменные:

```javascript
// Перед подключением modia/index.js
window.MODIA_LOG_ENABLED = false;     // Полная блокировка
window.MODIA_LOG_LEVEL = 'error';     // Только ошибки
```

### Приоритет активации

1. **ENV-конфигурация** (наивысший приоритет)
2. **GET-параметры** (?modia-log, ?modia-debug)
3. **Data-атрибуты** ([data-modia-log])

ENV-настройки переопределяют все остальные способы активации.

### Уровни логирования

| MODIA_LOG_LEVEL | Какие логи проходят |
|-----------------|---------------------|
| `'info'` | info, warn, error, success |
| `'warn'` | warn, error |
| `'error'` | error только |
| `undefined` | Все логи (по умолчанию) |
